Win your ticket to MLOC!

Intro

What is MLOC?

MLOC.JS 2014 is an international Javascript conference for people interested in the future of the language and the platform, the possibilities of JavaScript on the web and on the server side. For people doing or planning to do big JavaScript projects. And for people would like to see how is it possible to improve JavaScript by making it smarter, faster and sexy.

The game

Connect Four (also known as Captain's Mistress, Four Up, Plot Four, Find Four, Fourplay, Four in a Row and Four in a Line) is a two-player game in which the players first choose a color and then take turns dropping colored discs from the top into a seven-column, six-row vertically suspended grid. More info at: http://en.wikipedia.org/wiki/Connect_Four

Description

You should implement the Connect Four game using Node.js streams. After cloning/forking our repo, you will find the the test cases, which your solution must pass. The tests can be ran with npm test ( continuously with npm test-bg). Please use our template file (cf.js) for the implementation.

Choosing the winner

The shortest code wins (measured in bytes). If there will be two same-length-solution, the submission date will decide who wins. (You can find out more about making your javascript shorter on 140bytes).

Submission

Send your solution's secret gist's link to info@nodebp.com . Please only use the core stream module. The lesser the file size, the better! (just pass all the tests)

Deadline

2th, February, 11:59PM

Prices

The winner gets the MLOC ticket (accommodation, travelling NOT included), runner-ups will be awarded with WebStorm licences.

Questions

Please feel free to ask us anything! The preferred way of communication is through our twitter channel: @nodebp
