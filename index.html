Win your ticket to MLOC!

Intro

Connect Four (also known as Captain's Mistress, Four Up, Plot Four, Find Four, Fourplay, Four in a Row and Four in a Line) is a two-player game in which the players first choose a color and then take turns dropping colored discs from the top into a seven-column, six-row vertically suspended grid. More info at: http://en.wikipedia.org/wiki/Connect_Four

Description

You should implement the Connect Four game using Node.js streams. After cloning/forking our repo, you will find the the test cases, which your solution must pass. The tests can be run with npm test (you can continuously running by npm test-bg). Please use our template file (cf.js) for the implementation.

Choosing the winner

The shortest code wins (measured in bytes). If there will be two same-length-solution, the submission date will decide who wins. (You can find out more about making your javascript shorter on 140bytes).

Submission

Send your solution to info@nodebp.com, as a solution.js attachment. Please only use the core stream module. the lesser the file size, the better! (just pass all the tests)
